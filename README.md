# Telegram Todo-бот

Персональный бот для управления задачами прямо в Telegram. Добавляй, выполняй и удаляй задачи через удобные кнопки.

## Требования

- Python 3.8+
- pip
- Telegram-аккаунт

## Установка

1. Скачай или клонируй проект:
   ```bash
   git clone <url>
   cd todo-bot
   ```

2. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Создание Telegram-бота

1. Найди в Telegram: `@BotFather`
2. Напиши `/newbot`
3. Придумай имя и username для бота
4. Скопируй полученный **TOKEN**

## Настройка

1. Скопируй файл окружения:
   ```bash
   cp .env.example .env
   ```

2. Открой `.env` и вставь свой токен:
   ```
   TELEGRAM_TOKEN=1234567890:ABC-твой-токен-здесь
   ```

## Запуск

```bash
python main.py
```

Бот запустится и будет ждать сообщений. Логи пишутся в `logs/bot.log`.

## Команды

| Команда | Описание |
|---|---|
| `/start` | Приветствие и главное меню |
| `/add` | Добавить новую задачу |
| `/list` | Все задачи с кнопками управления |
| `/active` | Только невыполненные задачи |
| `/completed` | Только выполненные задачи |
| `/stats` | Статистика (всего / активных / выполненных) |
| `/clear` | Удалить все задачи (с подтверждением) |
| `/help` | Справка по командам |
| `/cancel` | Отменить текущее действие |

## Пример использования

1. Запусти `/start`
2. Нажми `/add` → введи "Купить молоко" → задача добавлена
3. Нажми `/list` — увидишь все задачи с кнопками
4. Нажми ✅ **Выполнить** под задачей — статус изменится сразу
5. Нажми `/stats` — посмотри сколько задач выполнено
